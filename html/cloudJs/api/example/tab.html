tab选项卡组件

简单的选项卡组件

cloudjs(selector).tab([options])

属性

属性名 类型 默认值 说明
axis string x 选项卡轴向。示例2
enableHover boolean false 鼠标挪到tab上是否会触发tab切换。示例3
hoverDelay number 200 在允许hover切换的情况下，鼠标挪到tab上触发选中的延迟时间。示例3
selectIndex number 0 默认选中项的索引。示例2
isAuto boolean false 选项卡是否按自动按顺序切换。示例4
inteval number 200 自动切换的时间间隔。示例4
onChange function 切换后的回调函数，参数默认带切换后的选项卡索引。示例4
