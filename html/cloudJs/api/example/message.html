message消息组件

cloudjs(selector).message([options]) options={'属性名':'属性值',...}

属性

属性名 类型 默认值 说明
type string success 消息提示类型，分为'success','warn','error'
content string 提示信息 消息正文
duration number 3000 延迟自动消失的时间，若为0，则默认不自动消失
isShowIcon boolean false 设置消息组件左上角是否展示icon
relative string/jquery对象 document 消息框所在的容器
offsetLeft number 0 偏离容器中心的位移，若isFull属性为true，则此属性无效
offsetTop number 0 偏离容器顶部的位移
isFull boolean false 消息框宽度是否为100%
position string in_top 消息框和容器的相对位置，in_top为容器内顶部，up，down，left，right，分别表示容器上下左右位置，若满足以下两个条件之一：
1、isFull设置为true
2、relative未设置或设置为document
则此参数设置无效，默认用in_top
isTriggerClose string false 设置是否点击页面其他元素自动关闭消息框

cloudjs(selector).message('方法名', [args])

方法

方法名 参数 说明
close 对象 关闭指定id对应的提示消息框，对象的参数key为id，value可为字符串或数组，若无参数，则关闭全部提示消息框

示例

示例1：最简单的调用方式，只需传content参数
示例2：设置不同的type值，生成不同类型的消息提示框
示例3：设置duration属性，控制消息提示框的保留时间
示例4：设置isShowIcon属性，控制是否在消息组件左侧显示相应的小图标
示例5：设置ralative属性，改变提示框的所在的容器,可传id或class，或jquery对象
示例6：设置offsetLeft和offsetTop属性，改变消息提示框中心偏移量
示例7：设置isFull属性，在容器顶部占据全部宽度显示
示例8：生成message时候可以设置id值，生成之后可依据id值关闭指定的消息提示框,若不指定id，则关闭全部消息提示框
示例9：设置position属性，改变消息弹出框相对于指定元素的位置
示例10：设置isTriggerClose属性，点击页面空白处可关闭此对话框
