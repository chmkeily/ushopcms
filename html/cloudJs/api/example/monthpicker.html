monthpicker月组件

cloudjs.monthpicker([options])

属性

属性名 类型 默认值 说明
width number 220 月控件的宽度
beginYear number -40 示例2开始年份,为当前年份的前40年
endYear number 10 结束年份,为当前年份的后10年
placeholder string yyyy-mm-dd 示例3输入框中显示的placeholder信息,需浏览器支持
timeFormat string 示例4显示在输入框中的格式
minDate string null 示例5可选的最小月份,格式为'+1Y,-1M'.必须以逗号分隔.也可以是某一具体月:'2015,05'
maxDate string null 可选的最大月份,格式同上
multiSelect boolean false 示例8选择几个连续的月
startMonth string 示例8默认开始月份,必须和 endMonth配对使用,并且multiSelect
endMonth string 示例8默认结束月份,必须和 startMonth配对使用
onSelect function 示例6选择月份后触发的回调函数
onClose function 示例7关闭月控件后触发的回调函数,配合 update方法,可以实现联动日历功能

方法

方法名 参数 说明
update [options] 示例7月组件的刷新方法,与日历组件功能相同
