dialog弹出框组件

页面的弹出对话框

cloudjs(selector).dialog([options]) options={'属性名':'属性值',...}

属性

属性名 类型 默认值 说明
modal boolean false 是否模态窗口打开，如果是则弹出框后面会有一层遮罩，页面的内容会被遮住不可点
title string 系统提示 设置对话框标题
width number 220 设置对话框的宽
height number/string auto 设置对话框的高
buttons object null 对话框的按钮，对应的按钮设置为{“按钮名称”：“回调方法”，“按钮名称”：“回调方法”}
draggable boolean true 对话框是否可以拖动
resizable boolean true 对话框是否可以改变大小
animate boolean false 是否通过动画打开对话框
fullscreen boolean false 是否支持全屏模式
relative string/object 对话框的相对元素，不设置时默认在窗口中间打开
position string 对话框相对元素的位置，up、down、left、right、up_left、up_right、down_left、down_right
(分别表示dialog出现在元素的正上方、正下方、正左方、正右方、上方偏左、上方偏右、下方偏左、下方偏右)，只在relative有实际元素时生效
onCreate function() 在对话框刚刚创建时调用，该方法在对话框的生命周期中只会调用一次。
onOpen function() 在对话框打开时创建，对于被close的对话框，在通过动作open打开时，该方法也会调用
onClose function() 在对话框被关闭时调用，对于destroy对话框，onClose方法会发生在onDestroy之前
onDestroy function() 一个对话框被销毁时调用，该方法在对话框的生命周期中只会调用一次。
onResizeStart unction(event, ui) 开始改变对话框大小时调用的方法，该方法有两个参数，第一个是普通的Event对象，第二个是装载了当前对话框高度宽度的对象
onResize unction(event, ui) 改变对话框大小时调用的方法，该方法有两个参数，第一个是普通的Event对象，第二个是装载了当前对话框高度宽度的对象
onResizeStop unction(event, ui) 对话框大小改变结束时调用的方法，该方法有两个参数，第一个是普通的Event对象，第二个是装载了当前对话框高度宽度的对象
onDragStart function(event, ui) 开始拖动对话框时调用的方法，该方法有两个参数，第一个是普通的Event对象，第二个是装载了当前对话框高度宽度的对象
onDrag function(event, ui) 拖动对话框时调用的方法，该方法有两个参数，第一个是普通的Event对象，第二个是装载了当前对话框高度宽度的对象
onDragStop function(event, ui) 对话框拖动结束时调用的方法，该方法有两个参数，第一个是普通的Event对象，第二个是装载了当前对话框高度宽度的对象
onFullscreen function() 当全屏时触发的事件
onRecover function() 当从全屏还原时触发的事件

cloudjs(selector).dialog('方法名', [args])

方法

方法名 参数 说明
destroy 彻底销毁弹出框，不可再通过open打开
close 隐藏弹出框，可通过open方法再次打开，不会清除填写的内容
open 可选的{属性：值，属性：值，……}对象 与close功能相反，重新打开弹出框，并可通过带参数来重新设置dialog的各种参数。
addButton object 传入一个按钮对象，来动态给弹出框添加按钮，对象结构如{name:'保存',handler：function(){},index:0}，其中name是按钮名称，handler是点击按钮触发的方法，index是按钮插入的位置
removeButton string 传入按钮名称来从弹出框中移除按钮
disableButton string 传入按钮名称来禁用弹出框中的按钮（此时并不会删除按钮）
enableButton string 传入按钮名称来启用弹出框中的按钮（和disableButton是反作用关系）
hideButton string 传入按钮名称来隐藏弹出框中的按钮（此时并不会删除按钮）
showButton string 传入按钮名称来显示弹出框中的按钮（和hideButton是反作用关系）
setTitle string 传入title名称来动态设置弹出框的标题
alert object 模拟浏览器的alert弹框，简化了自己实现一个模拟浏览器alert框的过程，参数对象可以设置3个属性：content、relative、position，分别表示：弹框内容、相对的元素、位置(up,down,left,right)
confirm object 模拟浏览器的confirm确认框，简化了自己实现一个模拟浏览器confirm确认框的过程，参数对象可以设置5个属性：content、relative、position、yes、no，分别表示：弹框内容、相对的元素、位置(up,down,left,right)、确定时调用的方法、取消时调用的方法
prompt object 模拟浏览器的prompt输入框，简化了自己实现一个模拟浏览器prompt输入框的过程，参数对象可以设置5个属性：content、relative、position、yes、require，分别表示：弹框内容、相对的元素、位置(up,down,left,right)、确定时调用的方法、是否必填(true,false)

浏览器的主要功能是将用户选择的web资源呈现出来，它需要从服务器请求资源，并将其显示在浏览器窗口中，资源的格式通常是HTML，也包括PDF、image及其他格式。用户用URI（Uniform Resource Identifier统一资源标识符）来指定所请求资源的位置。

　　

HTML和CSS规范中规定了浏览器解释html文档的方式，由W3C组织对这些规范进行维护，W3C是负责制定web标准的组织。

　　

HTML规范的最新版本是HTML4(http://www.w3.org/TR/html401/)，HTML5还在制定中（译注：两年前），最新的CSS规范版本是2（http://www.w3.org/TR/CSS2），CSS3也还正在制定中（译注：同样两年前）。
