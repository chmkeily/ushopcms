totop返回到顶部组件

cloudjs(selector).totop([options])

属性

属性名 类型 默认值 说明
bottom number 0 离屏幕底部的距离
right number 0 离屏幕右侧的距离
isScroll boolean true 是否滚动后才出现返回顶部图标
scrollHeight number 200 滚动到指定高度后才出现滚动条
img string ../../images/totop.png 返回到顶部的图片
hoverImg string hover状态下的图片

示例

示例1：默认调用，滚动条向下滚动大于200px出现，否则消失
示例2：始终固定在页面指定位置，并指定默认图片
示例3：指定默认图片和hover状态下的图片
