resizable缩放组件

让元素可通过鼠标缩放改变其宽高

cloudjs(selector).resizable([options])

属性

属性名 类型 默认值 说明
animate boolean false 缩放时是否有动画，详见示例9
animateDuration number/string slow 当设置了animate为true时，缩放动画的持续时间，可设置数字，单位ms，字符串可设置('slow', 'fast')，详见示例9
containment string/boolean false 局限在某个容器内缩放，详见示例2
maxHeight number null 缩放最大的高度，详见示例5
maxWidth number null 缩放最大的宽度
minHeight number 10 缩放最小的高度
minWidth number 10 缩放最小的宽度
aspectRatio number/boolean false 宽高缩放的比例，详见示例5
ghost boolean false 在缩放时，是否需要显示一个半透明的辅助元素，详见示例5
grid array/boolean false 缩放时的网格间距，默认是平滑的缩放效果，如设置为[3, 6]，则在x轴以每次3像素的间距缩放，在y轴以每次6像素的间距缩放
handles string e,s,se 该元素四边可缩放的方位(右、下、左、上，分别对应e、s、w、n，四个角分别用ne、se、sw、nw表示)，有8个值可组合选取(n、e、s、w、ne、se、sw、nw)，另可用all代表8个值全选，详见示例5
start function(event, ui) 每次开始缩放元素时触发, event为Event类型，ui为Object，可通过ui.size获取到元素的width和height，详见示例3
resize function(event, ui) 在缩放期间当鼠标移动时触发, event为Event类型，ui为Object，可通过ui.size获取到元素的width和height，详见示例3
stop function(event, ui) 每次停止缩放时触发, event为Event类型，ui为Object，可通过ui.size获取到元素的width和height，详见示例3

cloudjs(selector).resizable('方法名', [args])

方法

方法名 参数 说明
destroy 无 完全移除缩放功能，会把元素返回到它的预初始化状态，详见示例4
disable 无 禁用 resizable，这会阻止元素的缩放功能，除非是destroy后重新绑定缩放，否则只能通过enable来再次启动缩放功能 ，详见示例4
enable 无 启用 resizable，和disable类似一对开关，详见示例4
option 由属性和属性值组成的键值对 动态设置元素的缩放属性，详见示例5
