图片轮播组件

让你的图片展示得更优雅

cloudjs(selector).slide([options]) options={'属性名':'属性值',...}

属性

属性名 类型 默认值 说明
direction string x 轮播的方向,可选'x','y',分别表示延x轴轮播和延y轴轮播
duration number 500 每次轮播动画持续的时间
effects string slide 轮播动画效果，可选'slide'，'fade'，'cutout'
isAuto boolean true 是否自动轮播
order string forward 自动轮播的顺序，可选'forward'和'reverse',分别表示正向轮播和逆向轮播
interval number 5000 每次轮播的时间间隔
isCycle boolean true 是否允许循环轮播
startIndex number 0 开始的索引位置
isShowAnchor boolean true 是否展示锚点
anchorOffsetLeft number 锚点偏移左部的距离
anchorOffsetTop number 锚点偏移顶部的距离
isShowArrow boolean true 是否展示向前向后箭头
onChange function 每次轮播之后触发的函数，参数为当前索引位置。

cloudjs(selector).slide('方法名', [args])

方法

方法名 参数 说明
toNext 轮播到下一页
toPrev 轮播到上一页
toIndex number 轮播到指定的索引位置
