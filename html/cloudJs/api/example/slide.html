图片轮播组件

让你的图片展示得更优雅

cloudjs(selector).slide([options])

属性

属性名 类型 默认值 说明
direction string x 轮播的方向,可选'x','y',分别表示延x轴轮播和延y轴轮播。示例2
duration number 500 每次轮播动画持续的时间。示例3
effects string slide 轮播动画效果，可选'slide'，'fade'，'cutout'。示例3
isAuto boolean true 是否自动轮播。示例4
order string forward 自动轮播的顺序，可选'forward'和'reverse',分别表示正向轮播和逆向轮播。示例4
interval number 5000 每次轮播的时间间隔。示例4
isCycle boolean true 是否允许循环轮播。示例4
startIndex number 0 开始的索引位置。示例2
isShowAnchor boolean true 是否展示锚点。示例5
anchorOffsetLeft number 锚点偏移左部的距离。示例5
anchorOffsetTop number 锚点偏移顶部的距离。示例5
isShowArrow boolean true 是否展示向前向后箭头。示例5
enableMouseEnter boolean true 鼠标移入是否暂停自动轮播。示例2
onChange function 每次轮播之后触发的函数，参数为当前索引位置。示例6

cloudjs(selector).slide('方法名', [args])

方法

方法名 参数 说明
toNext 轮播到下一页。示例7
toPrev 轮播到上一页。示例7
toIndex number 轮播到指定的索引位置。示例7

示例2：控制轮播方向，开始的索引位置，鼠标移入是否暂停自动轮播

设置不同的direction参数，控制不同的轮播方向，默认为'x'；设置startIndex参数，选择开始的索引位置，设置enableMouseEnter参数，控制鼠标移入是否暂停自动轮播

示例3：改变轮播动画的持续时间和动画效果

设置duration参数，可以改变轮播动画的持续时间；设置effect参数，可以改变轮播的动画效果

示例4：允许循环轮播,自动轮播，控制自动轮播的顺序，自动轮播的时间间隔

设置isCycle参数，控制是否允许循环轮播；isAuto参数，设置是否自动轮播；order参数，设置自动轮播的顺序；interval参数，设置自动轮播的时间间隔

示例5：控制是否展示锚点，向前向后箭头；控制锚点的位置和样式

设置isShowAnchor参数，控制是否展示锚点；设置isShowArrow参数，控制是否展示向前向后箭头；设置anchorOffsetLeft和anchorOffsetTop参数，控制锚点的位置；容器内 也可以使用class=".slide_nav"自定义锚点,设置slide_anchor的css值，可设置锚点的样式，设置slide_anchor_active的css值，可以改变选中时的样式。
