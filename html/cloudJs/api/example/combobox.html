下拉组件

cloudjs(selector).combobox([options]) options={'属性名':'属性值',...}

属性

属性名 类型 默认值 说明
modal boolean false 是否模态窗口打开，如果是则弹出框后面会有一层遮罩，页面的内容会被遮住不可点
title string 设置对话框标题
width number 220 设置对话框的宽
height number/string auto 设置对话框的高
buttons object null 对话框的按钮，对应的按钮设置为{“按钮名称”：“回调方法”，“按钮名称”：“回调方法”}
draggable boolean true 对话框是否可以拖动
resizable boolean true 对话框是否可以改变大小
animate boolean false 是否通过动画打开对话框
fullscreen boolean false 是否支持全屏模式
relative string/object 对话框的相对元素，不设置时默认在窗口中间打开
posiiton string 对话框相对元素的位置，up、down、left、top，只在relative有实际元素时生效
onCreate function() null 在对话框刚刚创建时调用，该方法在对话框的生命周期中只会调用一次。
onOpen function() null 在对话框打开时创建，对于被close的对话框，在通过动作open打开时，该方法也会调用
onClose function() null 在对话框被关闭时调用，一个destroy的对话框，onClose方法会发生在onDestroy之前
onDestroy function() null 一个对话框被销毁时调用，该方法在对话框的生命周期中只会调用一次。
onResize function() null 改变对话框大小时调用的方法，该方法有两个参数，第一个是普通的Event对象，第二个是装载了当前对话框高度宽度的对象
onFullscreen function() null 当全屏时触发的事件
onRecover function() null 当从全屏还原时触发的事件

cloudjs(selector).dialog('方法名', [args])

方法

方法名 参数 说明
destroy 彻底销毁弹出框，不可再通过open打开
close 隐藏弹出框，可通过open方法再次打开，不会清除填写的内容
open 与close功能相反，重新打开的弹出框会保持之前的内容，要通过代码去修改内容。
setData {'属性名':'属性值',...} 重新设置dialog的各种参数
