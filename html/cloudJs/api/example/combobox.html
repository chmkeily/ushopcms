下拉组件

cloudjs(selector).combobox([options])

属性

属性名 类型 默认值 说明
name string 保存value值的隐藏域的name值。示例3
data array/string 下拉面板的数据，可以是json数组格式，或者请求数据的url。示例5
comValue string value value字段名。示例5
comText string text text字段名。示例5
checkValue boolean false 是否检查value值必须是data里面的value。示例1
defaultValue string 默认选中选项的value值。示例4
isMulti boolean false 是否多选。示例4
width number 设置下拉面板的宽度，默认会与输入框宽度一致。示例3
height number 200 设置下拉面板的高度。示例3
placeholder string 输入提示信息。示例5
separator string ; 多选时，选中值的分隔符，必须是data里面不存在的符号。示例4
showAll boolean true 是否显示全部数据。示例10
recordLen number 10 showAll为false时，显示的记录数。示例10
showIcon boolean true 是否显示下拉图标。示例8
checkAllText string 全选 多选下拉，“全选”文字。示例4
unCheckAllText string 取消选择 多选下拉，“取消选择”文字。示例4
showCheckAllText boolean true 是否显示“全选”文字。示例7
showCheckbox boolean true 多选的时候，是否显示复选框。示例8
params string/object 异步请求下拉数据参数，{%s}表示需要替换掉的字符串。示例8
onAjax function 处理异步请求结果，生成json或json数组格式数据，并return该数据。示例8
onCreate function 当创建下拉组件时触发的事件。示例6
onOpen function 当打开下拉面板时触发的事件。示例6
onClose function 当关闭下拉面板时触发的事件。示例6
onSelect function 当选中的值发生改变时触发的事件。示例6
onCheckAll function 当全选时触发的事件。示例6
onUnCkeckAll function 当取消选择时触发的事件。示例6

cloudjs(selector).combobox('方法名', [args])

方法

方法名 参数 说明
getValue 获取当前选中的值。示例7
setValue string 需要设置选中数据的value值，多选下拉，多个值可以用设置的separator连接。示例7
addData array/json 跟初始设置的data数据格式保持一致。示例7
delData string 需要删除的数据的value值，多个值可以用设置的separator连接。示例7
refresh array 刷新下拉列表数据，必须是json数组。示例7
enable 启用下拉组件。示例7
disable 禁用下拉组件。示例7
