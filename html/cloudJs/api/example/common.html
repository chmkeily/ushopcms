cloudJs公共方法

调用方式：cloudjs.util.方法名(args)

方法名 参数 功能说明
isEndWith string,string 判断字符串str是否以某字符串start开始
isStartWith string,string 判断字符串str是否以某字符串start开始
indexOf string|array, point, fromIndex 兼容低于IE9版本浏览器中Array的indexOf方法，使用此方法可以查找数组或字符串，point为查找目标，fromIndex表示查找的起始点
log *(任意参数和类型) 解决了console.log在IE6、7下会报错的问题。作用和浏览器默认的console.log一致，把传入的值一一打印出来
error *(任意参数和类型) 解决了console.error在IE6、7下会报错的问题。和浏览器默认的console.error一致，把传入的值一一设置成错误标志并打印出来
trace *(任意参数和类型) 解决了console.trace在IE6、7下会报错的问题。和浏览器默认的console.trace一致，把传入的值一一打印出来，并能打印出调用栈
