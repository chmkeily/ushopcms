draggable拖动组件

让元素可通过鼠标拖动改变其位置

cloudjs(selector).draggable([options])

属性

属性名 类型 默认值 说明
containment string/boolean false 局限在某个容器内拖动，详见示例2
cursor string auto 设置拖动时的鼠标样式，详见示例5
opacity number/boolean false 当被拖动时不透明度，详见示例5
axis string/boolean false 指定拖动时只能按在某个轴方向上拖动，('y'、'x')，详见示例5
grid array/boolean false 拖动时的网格间距，默认是平滑的拖动效果，如设置为[3, 6]，则在x轴以每次3像素的位移拖动，在y轴以每次6像素的位移拖动，详见示例5
handle string/element/boolean false 在该元素上的一个拖动把手，鼠标只有在该把手上才能拖动元素，把手必须是该元素内的节点。如一个对话框，只想鼠标在对话框头部时才能拖动整个对话框，则需要设置handle为头部元素，详见示例5
create function(event) 当元素被初始化为可拖动时触发, event为Event类型，详见示例2
start function(event, ui) 每次开始拖动元素时触发, event为Event类型，ui为Object，可通过ui.position和ui.offset获取到当前的css位置和偏移值，详见示例3
drag function(event, ui) 在拖动期间当鼠标移动时触发, event为Event类型，ui为Object，可通过ui.position和ui.offset获取到当前的css位置和偏移值，详见示例3
stop function(event, ui) 每次停止拖动时触发, event为Event类型，ui为Object，可通过ui.position和ui.offset获取到当前的css位置和偏移值，详见示例3

cloudjs(selector).draggable('方法名', [args])

方法

方法名 参数 说明
destroy 无 完全移除拖动功能，会把元素返回到它的预初始化状态，详见示例4
disable 无 禁用 draggable，这会阻止元素的拖动功能，除非是destroy后重新绑定拖动，否则只能通过enable来再次启动拖动功能 ，详见示例4
enable 无 启用 draggable，和disable类似一对开关，详见示例4
option 由属性和属性值组成的键值对 动态设置元素的拖动属性，详见示例4
