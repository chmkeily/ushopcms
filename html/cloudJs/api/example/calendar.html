calendar日历组件

cloudjs.calendar([options])

属性

属性名 类型 默认值 说明
months number 2 示例2同时显示的月份数
width number 220 单个月时日历控件的宽度
beginYear number -40 示例3开始年份,为当前年份的前40年
endYear number 10 结束年份,为当前年份的后10年
placeholder string 示例4输入框中显示的placeholder信息,需浏览器支持
timeFormat string yyyy-mm-dd 示例5显示在输入框中的格式,默认为'yyyy-mm-dd'
minDate string null 示例6可选的最小日期,格式为'+1Y,-1M,+5D'.必须以逗号分隔.也可以是某一具体日期:'2015,05,12'
maxDate string null 可选的最大日期,格式同上
multiSelect boolean false 示例9选择一段时间,并显示在同一个输入框中
startDate string 示例9时间段的默认开始日期,必须和 endDate成对使用
endDate string 时间段的默认结束日期,必须和 startDate成对使用
onSelect function 示例7选择日期后触发的回调函数
onClose function 示例8关闭日历控件后触发的回调函数,配合 update方法,可以实现联动日历功能

方法

方法名 参数 说明
update [options] 示例8日历控件的刷新方法,调用此方法可以刷新日历的默认设置
