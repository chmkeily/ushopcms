cookie浏览器缓存组件

cloudjs.cookie('方法名', [args])

方法

方法名 参数 说明
setCookie 对象
{ name:'', value:'', expires:0, allPath:false }		 设置cookie
name:cookieName，必填
value:cookieValue，可以是字符串、数组或对象，必填
expires:有效天数，默认0表示永久（直到清空浏览器缓存）,默认为0可省略
allPath:为false表示同目录下的文件都可以获取该cookie,true表示同一主机名下文件都可以获取该cookie，默认为false可省略
getCookie cookieName,回调函数 获取指定cookie值,字符串
getCookieObj cookieName,回调函数 获取指定cookie值，把原来存储的值转换成对象返回
removeCookie cookieName 删除指定cookie
