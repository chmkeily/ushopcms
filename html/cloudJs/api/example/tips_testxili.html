tips提示组件

用于页面元素的信息提示

cloudjs(selector).tips([options])

属性

属性名 类型 默认值 说明
track boolean false 设置提示信息是否跟随鼠标移动，默认不随鼠标移动，详见示例2
title string/function(ele, callback) 设置提示的信息文字，如果被作用的元素本身也有title属性值，则优先使用设置的title属性值，如果两者都没有设置title，则不会出现提示信息，在这里设置的title也可以是一个方法，该方法会传入一个callback参数，调用方将要设置的title传入callback即可，详见示例1
position string 设置提示信息出现的方位，有left、right、up、down、down_right、down_left、up_right、up_left八种，如果不设置，则按照组件自动计算的方位，当你设置的position会导致tips超出边界的时候，组件也会自动调整position，详见示例2
hover boolean false 是否支持鼠标移到提示信息上时，提示信息不消失，方便拷贝，详见示例4
hideOnFocus boolean true 鼠标单击目标元素时，提示信息是否消失（此时鼠标还在目标元素区域内），详见示例9
isContainer boolean false 是否容器，当为容器时，内部的包含有title的元素也会被设置作用到tips，详见示例7
loading string loading title为回调函数时显示加载的文本内容，详见示例8
showDelay number 200 鼠标移动到目标元素后tips多长时间内出现，默认是200表示延迟200ms出现，详见示例6
hideDelay number 300 当鼠标离开目标元素后tips多长时间后消失，默认是300表示300ms后消失，详见示例6
width number 300 设置tips的宽，详见示例10
height number 150 设置tips的高，当内容超出设置的宽高的时候，tips将会出现滚动条，详见示例10

cloudjs(selector).tips('方法名')

方法

方法名 参数 说明
setData 重新设置属性值，分永久设置和临时设置，永久设置会作用于后面重新弹出的提示信息，而临时设置作用于当前弹出的提示信息，详见示例9
destroy 销毁tips，可通过传入isContainer来设置是否销毁容器内的所有tips，详见示例10
